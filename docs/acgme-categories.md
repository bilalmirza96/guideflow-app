# ACGME Defined Category Minimums — General Surgery

Reference data for the case log module. These minimums are used to calculate graduation progress bars and category-level compliance.

## Graduation Requirements

- **Total major cases as Surgeon (SC + SJ):** 850
- **Chief cases (SC):** ≥ 200 of the 850
- **PGY-3 cutoff:** 250 cases (as OS or FA) before starting PGY-3, ≥ 200 in defined categories
- **Teaching Assistant (TA):** ≥ 25 cases (PGY-4/5 only; count toward total but NOT toward 200 Chief minimum)

## Zwisch Scale — Operative Autonomy

The Zwisch scale is used to document the level of operative autonomy given to a resident during a procedure. Resident self-reports this level at case submission. Attending confirms or adjusts via EPA assessment.

| Level | Label | Meaning |
|---|---|---|
| 1 | Show and Tell | Attending performs, resident observes |
| 2 | Active Assistance | Resident assists with significant attending guidance |
| 3 | Passive Help | Resident performs with attending providing minor corrections |
| 4 | Supervision Only | Resident performs independently, attending present but not scrubbed |
| 5 | Solo | Resident performs without attending in room |

**Assessment Process:** Resident self-reports autonomy level at case submission. Attending receives EPA assessment request with mobile magic link (72-hour expiry) and confirms or adjusts the autonomy level. If attending does not respond within 72 hours, resident self-report stands.

## ABS EPA Entrustment Scale (1–5)

The Entrustable Professional Activity (EPA) assessment uses a five-point scale across three surgical phases: Pre-operative / Intra-operative / Post-operative. Each phase is scored independently by the attending.

| Score | Label | Description |
|---|---|---|
| 1 | Not yet assessable | Has not had sufficient opportunity |
| 2 | Requires direct supervision | Performs with significant guidance |
| 3 | Requires indirect supervision | Performs with attending available but not directly involved |
| 4 | Independent | Performs independently in routine situations |
| 5 | Aspirational | Can supervise others performing this activity |

**Assessment Format:** Post-case EPA modal appears after resident submits operative case log. Attending receives single-use magic link and completes mobile-optimized form for Pre-op, Intra-op, and Post-op phases. Attending also confirms or adjusts the resident's self-reported Zwisch autonomy level.

## Fellowship-Specific Case Categories

Residents select a fellowship goal during onboarding. GuideFlow tracks progress against the case category minimums and research output typical of applicants who matched into that fellowship. This enables personalized fellowship readiness tracking and gap analysis.

| Fellowship | Society | Key Case Categories |
|---|---|---|
| Colorectal Surgery (CRS) | ASCRS / Fellowship Council | Large Intestine, Anorectal, Laparoscopic Complex |
| Hepatobiliary (HPB) | AHPBA / Fellowship Council | Biliary, Liver, Pancreas |
| Minimally Invasive (MIS/Bariatric) | SAGES / Fellowship Council | Laparoscopic Basic, Laparoscopic Complex, Upper Endoscopy |
| Surgical Oncology | SSO / ACS | Breast, Head & Neck, Skin/Soft Tissue, Abdominal |
| Vascular Surgery | SVS / ABS | Vascular — Access, Anastomosis/Repair |
| Thoracic Surgery | STS / AATS | Thoracic, Thoracotomy |
| Breast Surgery | ASBrS | Breast — Mastectomy, Axilla |
| Pediatric Surgery | APSA / ABS | Pediatric Surgery |
| Surgical Critical Care | SCCM / ABS | Surgical Critical Care, Trauma, Bedside Procedures |
| Transplant Surgery | ASTS / AAAT | Liver, Pancreas, Vascular Access |

**Readiness Tracking:** Fellowship readiness score is calculated by comparing resident's case counts in relevant categories against matched applicant benchmarks. Gap analysis includes:
- Case count vs. matched applicant average by fellowship and category
- EPA entrustment level on fellowship-relevant procedures
- Research output (publications, presentations) vs. matched applicant average
- Rotation profile completeness vs. matched applicants
- Application timeline and projected readiness date
